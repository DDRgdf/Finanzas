# Advanced Financial Market Analyzer

A comprehensive financial market analysis tool built with Python, featuring real-time tracking, technical analysis, and AI-powered predictions.

## Features

- Real-time market data tracking and visualization
- Technical analysis with multiple indicators (RSI, MACD, Bollinger Bands, etc.)
- Advanced price predictions using LSTM and ensemble models
- Interactive charts and customizable displays
- Risk analysis and trading recommendations
- Fundamental analysis integration
- Custom financial assistant for personalized advice

## Installation

### Prerequisites

Make sure you have Python 3.8+ installed on your system.

### Required Libraries

Install the required packages using pip:
bash
pip install yfinance
pip install pandas
pip install numpy
pip install matplotlib
pip install seaborn
pip install tensorflow
pip install scikit-learn
pip install ta
pip install plotly
pip install xgboost
pip install optuna
pip install streamlit

2. Enter a stock symbol (e.g., AAPL, MSFT, GOOGL)

3. Access different analysis tools:
   - Real-time tracking
   - Technical analysis
   - Price predictions
   - Risk analysis

## Main Components

- `gui.py`: Main application interface
- `real_time_window.py`: Real-time market data tracking
- `prediction_window.py`: Price prediction interface
- `New_Text_Document.py`: Advanced prediction models and analysis
- `ayud_fin.py`: Financial assistant and recommendation engine
